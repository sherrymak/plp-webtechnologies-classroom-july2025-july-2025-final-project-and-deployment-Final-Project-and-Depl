# FGCK Mutonya Church Website

A website for **FGCK Mutonya Church** is both a ministry hub and a church management platform, designed to:to inform, connect, and spiritually support members and visitors.  
The site showcases church ministries, sermons, events, and contact information, providing an engaging and user-friendly experience.

---

## **Website Purpose**

- Serve as an **information hub** for church members and visitors  
- Showcase church **ministries**: Youth, Women, Men, Children  
- Share **baily bible verses,sermons and teachings**  
- Provide **contact and outreach** options for prayer requests and inquiries  

---

## **Pages**

1. **Home** – Welcome message, latest sermons, ministry previews, call-to-action buttons  
2. **About** – Church history, mission, vision, Pastor’s message, core beliefs  
3. **Ministries** – Grid of ministries with details and meeting times  
4. **contact** – a foam where anyone can reach out  
5. **admin** – private sector for admin to login into the admin dashboard that showcase members, church assets, sermons and other church accounting elements.  

---

## **Internal Navigation (User Journey)**

### **Main Menu (Top Navigation)**
- Home  
- About  
- Ministries  
-  contact
- admin  

### **Home Page Links**
- **About Page** via “Learn More” or Pastor’s Message  
- **Ministries Page** via ministry cards  
- **admin Page** via “admin log in 
- **Contact Page** via call-to-action buttons or footer links  

### **About Page Links**
- Back to **Home** via menu  
- Link to **Ministries Page** (Learn More button)  
- **Contact Page** via footer  

### **Ministries Page Links**
- Each ministry card links to its **full ministry details page**  
- Header navigation links back to **Home**  
- Footer links to **Contact Page**  

### **Sermons Page Links**
- Each sermon links to a **sermon detail page**  
- Header links back to **Home**  
- Footer links to **Contact Page**  

### **Contact Page Links**
- Header links to all main pages  
- Footer repeats main navigation for easy access  
- Embedded map links to location directions  

---

## **Features**

- Responsive design for desktop and mobile  
- Interactive ministry cards linking to detailed pages  
- Latest sermons section on Home page  
- Contact form with direct email submissions  
- Footer with navigation links and church contact info
